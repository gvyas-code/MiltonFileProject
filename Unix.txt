#Command:grep
1.Grep stands for Global Regular Expression parser
2.Use to Search data in a file
3.Here we will cover search data in ine file
Search 1: Search data in a file
eg. grep "content" filename

Search 2: Search data in multiple file in a location
eg. grep "content" filename1 filename2 filename3

Search 3: Search data in all files in a location
eg. grep "content" *.*

Search 4: Search data with case insensitive
eg. grep -i "data" files

Search 5: Search data and display line number
eg. grep -n "data" filename

Search 6: Search data and display filename where it found
eg. grep -l "Text" *.*

Search 7: Search data by matching exact word(Not as a substring)
eg. grep -w "Text" Filename

Search 8: Search data by ignore (exclude)given data
eg. grep -v data Filename

Search 9: Search data and display only count of matches
eg. grep -c Text File

Search 10: Search data and display result.Also display surrounding lines of matched result

eg.
grep -B 5 Text Filename
grep -A 5 Text Filename
grep -C 5 Text Filename

ADVANCE SEARCH

Search 11: Search multiple data and display result
eg.
grep -e "FirstData" -e "SecondData" Filename 


#Command: egrep(EXTENDED GREP)

Search 12: Search multiple data and display result
eg. egrep -n -i "She|the|2010" filename

Symbol to use in commands
> :- redirects standard output

head -n 23 file.txt | tail -n 8 | grep 2009

Compare Files Command

Command 1: cmp
-Compare two files byte by byte 
-Display first different byte
Options
-Compare only given number of bytes(-n)
-Print only different bytes
-Display clear message in output (verbose)(-l)
eg. cmp file1 file2 
cmp -n file1 file2
cmp -l file1 file2


Command 2: comm
-compare sorted files line by line
-Display 3 columns in output
Output(-1,-2,-3)
- column 1 : content in file 1
- column 2 : content in file 2
- column 3 : content in file 3
Display only selected columns

comm -1 -2 file1 file2 //remove column 1 & 2 

Command 3: diff

#Cut command-Process data of a file
Use to fetch data from top to down(column data)
Works only on files having column formatted data
Syntax-1 Display only a particular position character(eg. cat filename, cut -c1 filename) // d-delimeter
Syntax-2 Fetch data from range of characters(cut -c1-3 filename, cut -c3- filename, cut -c-3 filename)

Syntax-1 Display only a particular column data(eg. cat filename, cut -d "," -f1 filename) //
Syntax-2 Fetch data from range of column(cut -d "," -f1-3 filename, cut -d "," -f3- filename, cut -d "," -f-3 filename)


Syntax:-Display all column by skipping only few columns(eg. cat filename, cut -d "," --complement -f1 filename, cut -d "," --complement -f3-5 filename,
cut -d "," --output-delimeter "|" -f3-7 filename) //
Option:-Cut command with only column formatted data(cut -s -d "," -f3-5 filename)






#SED:-Stream Editor
Used to Display and Editing Data
Editing Opetions are -Insertion | Addition | Deletion
2 Types Operation:-
-Line Addressing
Command 1: Display line multiple times(eg. cat filename, sed '3p' filename)
Command 2: Display only specific line(eg. cat filename, sed -n '3p' filename)
Command 3: Display last line(cat filename,sed '$p' filename, sed -n '$p' filename)
Command 4: Display Range of Line(eg. cat filename, sed -n '1,3p' filename)
Command 5: Do not display specific lines(eg. cat filename, sed -n '2!p' filename)
Command 6: Do not display specific range of lines(eg. cat filename, sed -n '2,4!p' filename)
-Context Addressing
